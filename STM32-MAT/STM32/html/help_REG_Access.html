REG Access block

Block used in the model to generate a line of C code for STM32 register peripheral access.

Depending on STM32CubeMx configuration.

Contents

  • S-Function
  • Register Access parameters
  • Peripheral
  • Read
  • Write

S-Function

Nb Input: 0

Nb Output:

  • 0 for Write operation
  • 1 for Read operation

Register Access parameters

Peripheral selection depending on STM32CubeMX configuration.

Text information for C code generation

Peripheral

Peripheral selection from configured peripheral list. Only peripherals configured using STM32CubeMX are available.

top

Read

Full Register Name (mandatory) is exact C code text for register read access. It is based on HAL library

Comments (optional) is comment to be included before C code generated line.

Register value available as "Register access" S-Function output.

C generated code example :

top

Write

Full Register Name (mandatory) is exact C code text for register write access. It is based on HAL library

Full Value (mandatory) is exact C code text for register value to write. It is based on HAL library

Operation (mandatory) is C code register operating value to apply.

Comments (optional) is comment to be included before C code generated line.
