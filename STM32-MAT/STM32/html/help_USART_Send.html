S-Function

Nb Input: 2

  1. Nb2Send : Number of char to send.
  2. Data2Send : Buffer address of data to send.

Nb Output: 1

NbSent : Number of data sent.

USART

Usart information

Usart Name

Set USART name. Only STM32CubeMx configured USART/UART are available.

Send Buffer Size

Ring buffer size used to send messages.

Parameters

STM32CubeMx Usart configuration

Mode

Read only parameter value from STM32CubeMx : Synchronous or Asynchronous mode.

DMA RX

Read only parameter value from STM32CubeMx : Use DMA for input communication.

DMA TX

Read only parameter value from STM32CubeMx : Use DMA for output communication.

Interrupt

Read only parameter value from STM32CubeMx : Use Rx and TX Interrupt.
