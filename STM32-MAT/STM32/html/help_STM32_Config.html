STM32 configuration selection model

This block is used to select STM32 configuration done with STM32CubeMx. This block is mandatory. It gives the STM32 reference and drivers that will be able to use for this design.

S-Function

No Input and No output

top

Block Parameters

To select the ioc file used for this design. The ioc file is the file that gathers all the information and configuration on the STM32 made from STM32CubeMx.

top

STM32 configuration file path

The path of ioc file. Relative to slx model path.

top

Configuration file

The name of ioc file selected for this design.

top

Select STM32 configuration file

Opens a browser to select path and name of ioc file used for this design.

top

Start STM32CubeMx configuration tool

Opens STM32CubeMx configuration tool.

It does not change the selected ioc file for this design.

  1. Current ioc file selection: STM32CubeMx shows configuration stored on the selected ioc file.
  2. New ioc file selection: STM32CubeMx opens without any selected project.
  3. Browse for ioc file selection: Opens a browser to select ioc file to open with STM32CubeMx.

top

MCU Name

STM32 currently configured in the selected ioc file.
