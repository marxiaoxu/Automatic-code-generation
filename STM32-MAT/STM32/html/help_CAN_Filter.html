CAN Filter parameters

top

CAN Name

Name of CAN communication channel selection (CAN or CAN1/CAN2 depending on STM32CubeMx configuration).

Filter Activation

Filtering parameters are taken into account when it is activated.

Filters not used by the application should be left deactivated.

FIFO

The FIFO associated to the reception filter (CAN_FIFO0 or CAN_FIFO1).

Filter Mode

Can be MASK or LIST. To filter a group of identifiers, configure the Mask/Identifier registers in mask mode.

To select single identifiers, configure the Mask/Identifier registers in identifier list mode.

Filter Scale

Filter Id are 16/32 bit.

  • mapping: ------------------------------------------------------- 31XXXXXX24 ------ 23XXXXXX16
  • 32-Bit filter - Mask&List IDH[31:16] ------ map -------- STID[10:3] ------- STID[2:0]+EXID[17:13]
  • mapping: ------------------------------------------------------- 15XXXXXX8 ------- 7XXXXXX0
  • 32-Bit filter - Mask&List IDL[15:0] ------ map --------- EXID[12:5] ------ EXID[4:0]+IDE+RTR+0
  • 16-Bit filter - Mask&List IDH[15:8] ------ map --------- STID[10:3]
  • 16-Bit filter - Mask&List IDL[7:0] ------ map ---------------------------------- STID[2:0]+RTR+IDE+EXID[17:15]

Filter ID High/Low

In mask mode the identifier registers are associated with mask registers specifying which bits of the identifier are handled as “must match” or as “don’t care”.

In list mode, the mask registers are used as identifier registers. Thus instead of defining an identifier and a mask, two identifiers are specified, doubling the number of single identifiers. All bits of the incoming identifier must match the bits specified in the filter registers.

Filter Mask ID High/Low

In mask mode the mask registers is specifying which bits of the identifier are handled as “must match” or as “don’t care”.

In list mode, the mask registers are used as identifier registers.

Filter Number

Each filter within a filter bank is numbered (called the Filter Number) from 0 to a maximum dependent on the mode and the scale of each of the filter banks.

Bank Number

Up to 28 (0:27) or 14 (0:13) configurable and scalable filter banks.
