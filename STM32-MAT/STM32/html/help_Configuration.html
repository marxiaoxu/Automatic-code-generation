STM32 Configuration model

Block used to set the STM32 configuration with the STM32CubeMx tool.

Nb : This block is mandatory for all STM32 Simulink application and must be used once.

Contents

  • S-Function
  • Parameters
  • STM32 configuration file path

S-Function

Nb Input: 0

Nb Output: 0

Parameters

top

STM32 configuration file path

Read only parameter value : Path to STM32CubeMx selected configuration file.

Configuration file

Read only parameter value : Name of STM32CubeMx selected configuration file.

Select STM32 configuration file

Open browser to select STM32CubeMx configuration file.

Update STM32 configuration file path and Configuration file and MCU Name fields when (ioc) STM32CubeMx configuration file has been selected.

top

Start STM32CubeMx configuration tool

Start STM32CubeMx configuration tool to:

  • Modify Current ioc file : Open STM32CubeMx to modify selected Configuration file
  • Create New ioc file : Open STM32CubeMx without any selected configuration.
  • Browse for ioc file : open browser to select ioc file to modify. STM32CubeMx opens selected configuration.

top

MCU Name

Read only parameter value from STM32CubeMx : MCU selected in current configuration file
