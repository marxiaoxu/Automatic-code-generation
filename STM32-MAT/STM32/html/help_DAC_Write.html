DAC block

Block used in the model to select Digital to Analog Converter (DAC).

Depending on STM32CubeMx configuration.

Contents

  • S-Function
  • Parameters
  • OUTx

S-Function

Nb Input: 1 or 2

  • DACx (x=1..2) DAC input channel value.

Nb Output:

  • In case of with DMA use-case, 1 function call for IRQ Named: Tranfer Complete Half Transfer Complete Error Underrun

Parameters

DAC output selection.

top

OUTx

DACx (x=1,2) input model value is analog converted to OUTx (x=1,2) selected output.

DMA

No output when DMA is selected.

Enter DMA buffer sizes for dedicated outputs.

Data Store Memory must be defined to access to generated code for DMA buffer pointer.
