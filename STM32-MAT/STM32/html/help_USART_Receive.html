S-Function

Nb Input: 1

  • Nb2Rcv : model outputs are updated when Nb2Rcv bytes have been received.

Nb Output: 3

  1. NbRcv : Number of received bytes.
  2. PtRcv : Address of received buffer.
  3. dataRcv : Received buffer.

USART

Usart information

Usart Name

Set USART name. Only STM32CubeMx configured USART/UART are available.

Rcv Buffer Size

Maximum buffer size used to receive messages. Must be higher than Nb2Rcv input model value. (not too big to save memory)

Enable Software Interrupt

Selected : Then, software interrupt number can be selected. Software interrupt trig when Nb2Rcv bytes have been received.

Then, function call connected to Exti selected number of GPIO_Exti block is called. Like External interrupt selected number (GPIO port is not taken into account)

Not selected : No software interrupt.

Parameters

STM32CubeMx Usart configuration

Mode

Read only parameter value from STM32CubeMx : Synchronous or Asynchronous mode.

DMA RX

Read only parameter value from STM32CubeMx : Use DMA for input communication.

DMA TX

Read only parameter value from STM32CubeMx : Use DMA for output communication.

Interrupt

Read only parameter value from STM32CubeMx : Use Rx and TX Interrupt.
