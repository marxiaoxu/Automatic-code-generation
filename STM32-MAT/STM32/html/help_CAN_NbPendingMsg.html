CAN Pending Message block

Block used in the model to get the number of pending message from the receiving FIFO of the selected CAN.

S-Function

Nb Output: 1 Number of pending messages.

CAN Nb Pending Msg parameters

top

CAN Name

Name of CAN communication channel selection (CAN or CAN1/CAN2 depending on STM32CubeMx configuration).

Mode

STM32 can be Master or Slave depending on STM32CubeMx configuration.

Interrupt Enabled

Communication mode information that depends on STM32CubeMx configuration. Activated in case of interrupts.

FIFO

Used to select the FIFO (CAN_FIFO0 or CAN_FIFO1).
