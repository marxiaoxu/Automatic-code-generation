I2S_Stop

STM32 I2S driver based on the HAL library, to stop DMA.

Contents

S-Function

This model is used to stop the I2S Direct Memory Access.

No Input & No Output

Parameters

  1. I2Sx : I2S selection. Only I2S configured from STM32CubeMX are available.
